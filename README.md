# Harwinder-CSharp-Projects

This repository showcases my completed C# projects from The Tech Academy training program. Each project demonstrates key programming concepts and practical application of .NET technologies.

## Completed Projects
- **Abstract Class Assignment** – Demonstrates inheritance and abstraction.
- **Calling Methods** – Practice with method calls and parameter passing.
- **CarInsurance** – Console app calculating insurance quotes using conditional logic.
- **CodeFirstStudentApp** – Entity Framework Code‑First project creating a Student database.
- **Constructor Assignment** – Shows object initialization and constructor overloading.
- **Datetime Assignment** – Manipulates and formats date/time values.
- **Lambda Assignment** – Uses lambda expressions and LINQ queries.
- **Operator Assignment** – Demonstrates operator overloading.
- **Entity Framework Assignment** – Database interaction using EF6.
- **ServerTimeApp** – ASP.NET Razor Pages web app deployed to Azure.
